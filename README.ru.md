# AI Cycling Coach — Система учета и анализа велотренировок

Персональный ассистент для спортсменов-любителей, предназначенный для разбора результатов, планирования тренировок и улучшения физической подготовки с помощью локальных нейросетей.

## Цель проекта
Разработать программу учета и статистики велотренировок с разбором результатов и планированием программ по улучшению физической подготовки. Система позволяет пользователю вносить текущие тренировки и планировать расписание на спортивный год.

## Ключевые возможности
* **Учет тренировок**: возможность вносить, корректировать или удалять данные о выполненных занятиях.
* **Планирование**: создание тренировочных планов, постановка задач и планирование предстоящих тренировок.
* **AI-Анализ**: использование локальной нейросети для разбора тренировок и предоставления советов.
* **Статистика**: отображение данных по времени, усилиям и накопление статистики.
* **Работа с данными**: загрузка через CSV-файлы с перспективой интеграции со Strava.

## Технологический стек
* **Backend**: Python (FastAPI).
* **Frontend**: HTML5, Jinja2, JavaScript.
* **Database**: SQL (SQLite) для локального хранения данных.
* **AI Engine**: Поддержка локальных нейросетей для приватности данных.

## Roadmap (Планы по развитию)

### Краткосрочные задачи
* Оптимизация взаимодействия с ИИ-агентом: ускорение обработки истории сообщений.
* Улучшение системного ИИ промта
* Добавление минимального дизайна 
* Локализация: перевод интерфейса на английский язык.
* Визуализация накопленных данных в виде графиков

### Среднесрочные задачи
* Визуализация: создание графиков прогресса и зон интенсивности.
* Расширение функционала: поддержка беговых тренировок.
* Гибкость ИИ: возможность подключения к внешним нейросетям через API.
* Перенос деплоя на удаленный сервер
  

## Инструкция по установке

1. Склонируйте репозиторий на ваш локальный компьютер.
2. Установите необходимые зависимости:
   `pip install -r requirements.txt`
3. Убедитесь, что у вас запущена среда для работы локальных LLM (например, Ollama).
4. Запустите сервер разработки:
   `uvicorn app.main:app --reload`
5. Перейдите в браузере по адресу: `http://127.0.0.1:8000`
